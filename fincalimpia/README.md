# Finca Limpia single-page website

Static website prepared for:

`https://englishstudy.club/fincalimpia/`

## Files

- `index.html` — the page content.
- `styles.css` — visual design, layout and mobile responsiveness.
- `script.js` — mobile menu and WhatsApp/contact configuration.
- `assets/` — logo, favicon and social preview image.
- `site.webmanifest` and `robots.txt` — browser/search support.

## Contact configuration

The WhatsApp number is already configured as `+34 644 852 762`.

Open `script.js` only if you want to change the WhatsApp message later.

No email is shown in this version.

## How to publish under EnglishStudy.club

Upload the whole `fincalimpia` folder to the root of the existing website package/repository.

The final path should be:

`/fincalimpia/index.html`

Then the public URL should be:

`https://englishstudy.club/fincalimpia/`

## Notes

This is independent from the Aula de Inglés Nava pages. It uses relative paths, so it should not interfere with the existing website.
